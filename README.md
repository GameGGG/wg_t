# 搞着玩的命令行工具


## 安装
```shell
    npm install wg_t -g
```

## 使用

1. 翻译请使用

```shell
    // t 翻译的单词/短语/句子(支持中英文)
    t 这是一个好的命令行工具

    // 结果如下
    // google:  This is a good command line tool
    // baidu:  This is a good command line tool
    // youdao:  This is a good command line tools
```


2. 提问请使用

```shell
    // t ? [问题]
    t ? 什么是命令行工具

    // 结果为打开浏览器进行百度搜索该问题
```

3. 打开网址请使用

```shell
    // t 网址
    t http://www.baidu.com
    t www.baidu.com

    // 使用默认浏览器打开对应网址
```
